ℹ️ About AcadWrite

AcadWrite is a free, open-source academic writing assistant built with Python + Flask. It includes a rich text editor, PDF viewer, academic source crawler (Semantic Scholar, arXiv, Google Scholar, Emerald), synonym lookup, spell-checking, outline generation, mind mapping, and session management.

No LLM or AI models are used — all intelligence is rule-based and human-guided.
